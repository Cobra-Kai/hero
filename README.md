# Hero

## Building

If you do not have a configure script, create one:

	autoreconf -i -I /usr/local/share/aclocal

Then configure and build the system:

	./configure
	make

## Running

Keyboard settings:

~~~
ESC - exit
E - forward
S - turn left
D - backward
F - turn right
W - strafe left
R - strafe right
` - toggle text input
~~~
